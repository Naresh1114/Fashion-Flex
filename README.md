Fashion Flex – Online Clothing Rental Management System

Fashion Flex is a modern and user-friendly web application developed to simplify the clothing rental process for customers, store managers, and administrators. The system allows users to browse fashion products, rent outfits online, manage returns, and track rental activities efficiently through a centralized platform.

This project was developed using Python and the Django framework with a responsive frontend built using HTML, CSS, and JavaScript. Fashion Flex aims to provide a smooth and organized rental experience while reducing manual work in clothing rental stores.

✨ Project Highlights
Easy-to-use rental management system
Secure user authentication and login system

🧩 Modules in Fashion Flex
👤 User Module

The User Module allows customers to interact with the platform and rent clothing products online.

Features:
User Registration & Login
Browse available outfits and fashion products
Search clothing items
Rent products online
View rental history and order details
Manage profile information
🛠️ Admin Module

The Admin has complete control over the system and manages overall platform activities.

Features:
Manage stores and managers
Monitor user activities
View reports and rental records
Manage system operations
Control product categories and overall database
🏬 Manager Module

Managers are responsible for handling store operations and rental processes.

Features:
Add, edit, and delete products
Manage inventory and stock
Approve rental requests
Handle product returns
Update product availability
🏪 Store Module

The Store Module helps maintain store-related details and product tracking.

Features:
Store information management
Product availability tracking
Rental status monitoring

⚙️ Installation & Setup Guide

Follow the below steps to run the project on your local machine.

1️⃣ Clone the Repository
git clone https://github.com/your-username/fashion-flex.git
2️⃣ Navigate to Project Folder
cd fashion-flex
3️⃣ Create Virtual Environment
python -m venv venv
4️⃣ Activate Virtual Environment
Windows
venv\Scripts\activate
Mac/Linux
source venv/bin/activate
5️⃣ Install Required Packages
pip install -r requirements.txt
6️⃣ Apply Database Migrations
python manage.py migrate
7️⃣ Run the Development Server
python manage.py runserver
8️⃣ Open in Browser
http://127.0.0.1:8000/
